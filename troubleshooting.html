Recipe App Troubleshooting

Issue Detected

The application is having trouble starting because:

  • The commands json-server and react-scripts are not being recognized
  • Port 3001 might be in use by another process

Manual Starting Instructions

Step 1: Install Dependencies

Open a command prompt in the project directory and run:

npm install

Step 2: Start JSON Server

Open a command prompt in the project directory and run:

npx json-server --watch db.json --port 3001

You should see output indicating the server is running at http://localhost:3001

Step 3: Start React App

Open another command prompt in the project directory and run:

npx react-scripts start

This will start the React application at http://localhost:3000

Accessing the Application

Once both servers are running:

  • Access the React app at: http://localhost:3000
  • The JSON Server API is available at: http://localhost:3001

Common Issues

Port Already in Use

If you see an error like EADDRINUSE, it means the port is already being used by another process.

To fix this:

For Windows:
# Find the process using the port
netstat -ano | findstr :3001

# Kill the process (replace PID with the process ID from the previous command)
taskkill /F /PID PID

Missing Dependencies

If commands are not recognized, ensure all dependencies are installed:

npm install json-server react-scripts --save
